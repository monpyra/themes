## 📥 How to Use

To use an official theme, simply update your `monpyra.toml` file:

```toml
[site]
theme = "monpyra-theme-ampera"  # Replace with your desired theme
```

The Monpyra Engine will automatically fetch the theme during the build process if it's not present in your local themes/ folder.

## 🛠️ Theme Anatomy
​Every official theme in this repository follows the Monpyra Theme Standards:
- Responsive Design: Mobile-first approach using Tailwind CSS.
- SEO Ready: Built-in JSON-LD and Open Graph support.
- Lightning Fast: Zero heavy dependencies, optimized asset minification.
- Feature Rich: Native support for Live Search, Table of Contents, and Dark Mode.
